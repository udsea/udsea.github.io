The technical side of this site is for posts that deserve more room than a thread or a README update.

Expect things like:

- implementation notes after building or debugging something
- architecture tradeoffs that are easier to explain with examples
- practical write-ups that start from a real problem and end with a working fix

If a post saves me from relearning something six months later, it belongs here.
